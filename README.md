# App de Perguntas e Respostas - Flutter
Este projeto é um aplicativo de perguntas e respostas sobre Flutter, desenvolvido como meu primeiro projeto em Flutter, com foco no aprendizado dos conceitos fundamentais do framework e da linguagem Dart.

## Objetivo do Projeto
O objetivo principal deste projeto foi aplicar, na prática, os conceitos iniciais de Flutter através da construção de um app simples de quiz.

Com este projeto, busquei:

- Entender a estrutura básica de um app Flutter
- Trabalhar com widgets e composição de UI
- Criar interação com o usuário através de botões
- Controlar o fluxo de perguntas e respostas
- Exibir feedback ao usuário com base nas respostas selecionadas

Este projeto representa o início da minha jornada com desenvolvimento mobile usando Flutter.

## Funcionalidades
- Exibição de perguntas na tela
- Alternativas de resposta em formato de botões
- Validação da resposta selecionada
- Avanço para a próxima pergunta
- Sistema de pontuação por pergunta acertada
- Interface simples e intuitiva

## Conceitos Abordados
Durante o desenvolvimento do app, foram abordados os conceitos:

- Widgets básicos (Text, Column, ElevatedButton, etc.)
- StatelessWidget e StatefulWidget
- Gerenciamento de estado simples
- Funções e callbacks
- Listas e mapeamento de dados
- Tipagem e sintaxe do Dart
- Organização inicial do código

## Tecnologias Utilizadas
- Flutter
- Dart
- Visual Studio Code
- Android Studio para Emulação Android

## Como Executar o Projeto
1. Clone o repositório:
  ``git clone https://github.com/m3ndxs/projeto-perguntas.git``

2. Acesse a pasta do projeto:
   ``cd projeto-perguntas``

3. Instale as dependências no terminal, caminho raiz do projeto:
  ``flutter pub get``

4. Execute o aplicativo:
  ``flutter run``
   
Certifique-se de ter o Flutter configurado corretamente no seu ambiente.
